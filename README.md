# MyWeb Project

A modern web application built with React, Vite, TypeScript, and TailwindCSS.

## 🚀 Architecture & Tech Stack

This project is built using the following exact versions:
- **React**: `19.2.4` (including `react-dom@19.2.4`)
- **Vite**: `6.4.2` (with `@vitejs/plugin-react@4.7.0`)
- **TypeScript**: `6.0.2`
- **Tailwind CSS**: `3.4.19` (with `postcss@8.5.8` and `autoprefixer@10.4.27`)
- **Framer Motion**: `11.18.2`
- **Lucide React**: `0.446.0`
